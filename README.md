## 🌍 Overview

Sahayak AI is an AI-powered emergency response platform that helps users during disasters by providing:

- 🤖 AI-based disaster analysis
- 📍 Live GPS location
- 🏥 Nearby hospital recommendations
- 🌦 Live weather information
- 🚨 Risk assessment
- 🗺 Interactive emergency map
- 🌐 Multi-language support
- 🎤 Voice-based emergency reporting
- 🔔 Browser notifications

# 🚀 Installation

## Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

## Frontend

```bash
cd frontend-vite
npm install
npm run dev
```
